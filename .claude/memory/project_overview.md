Event Queue: guest list and door management platform for nightlife venues/events in Brazil. MVP focuses on promoter/host/manager workflow for guest lists and check-ins. Post-MVP expands into ticketing.

**Why:** Competing with high-fee ticketing platforms in Brazil.
**How to apply:** All feature work should align with the MVP design doc vision. Product decisions favor the Brazilian nightlife market context (WhatsApp notifications, CPF documents, Pix payments post-MVP).
