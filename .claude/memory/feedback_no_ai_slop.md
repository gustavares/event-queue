---
name: No AI Slop Aesthetics
description: Frontend design must be distinctive and creative — avoid generic AI-generated aesthetics, template looks, and safe defaults
type: feedback
---

Never produce generic "AI slop" frontend design. Every UI output must be distinctive and surprising.

**Why:** User wants the platform to have a unique visual identity, not look like another template app. Generic AI output (Inter font, purple gradients, predictable layouts) is explicitly unwanted.

**How to apply:**
- Typography: Use distinctive fonts (Space Grotesk, Outfit, or better). Never Inter, Roboto, Arial, system fonts.
- Color: Dominant colors with sharp accents. No timid palettes. No purple-gradient-on-white.
- Motion: Orchestrated page load reveals > scattered micro-interactions. CSS-only preferred, Motion library for RN.
- Backgrounds: Layered gradients, geometric patterns, atmosphere. No flat white.
- Layouts: Context-specific character. No cookie-cutter patterns.
- Draw from nightlife aesthetics, editorial design, IDE themes for inspiration.
